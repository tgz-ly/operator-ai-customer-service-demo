# 运营商智能客服助手 Demo

## 项目简介
本项目基于 Dify 搭建运营商业务智能客服 Demo，支持套餐咨询、宽带故障处理、投诉工单分类、账单查询与业务取消等场景问答。

## 技术栈
- Dify
- DeepSeek Chat
- Knowledge Base / RAG
- Prompt Engineering
- Demo 测试与优化

## 核心功能
1. 套餐办理问答
2. 宽带故障排查
3. 投诉工单分类
4. 账单查询说明
5. 业务取消失败处理
6. 多问题拆分与逐项回答

## 知识库文件
- 套餐办理FAQ.txt
- 宽带故障处理流程.txt
- 投诉工单分类规则.txt
- 账单查询与业务取消说明.txt

## 测试与优化
第一轮测试发现账单类问题资料不足，于是补充账单查询与业务取消说明文档。
第二轮测试发现模型会扩展短信指令、公众号入口、退款时限等内容，于是通过 Prompt 约束禁止编造。
最终 Demo 可以对多问题进行拆分、分类并逐条回答。

## 项目收获
通过本项目掌握了 Dify 应用创建、知识库配置、RAG 问答、Prompt 约束、客服类 Demo 测试与优化流程。
